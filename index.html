Selectors in CSS.
Selector Syntax Description
Type Selector < element_name > The CSS type selector matches elements by node name. In other words, it selects all elements of the given type within a document.
Class Selector .class_name Selects all elements that have the given class attribute. (To know more about Class Selector, Click Here)
ID Selector #id_name Selects an element based on the value of its id attribute. There should be only one element with a given ID in a document.
Universal Selector * { /* rule */ } for changing anything in the whole HTML file.
Pseudo Class selector Selector : Pseudo-Class It is use to change the state of an element.
Pseudo element Selector selector ∷ pseudo-element-selector It is use to select element's pseudo elements.
Attribute Selector a [ href = "www.example.com"]{ /* Rules */} Use to select an anchor tag and style them
Combinator Selector Element ⟩ Child Use to select child elements from their parent elemnt.
Combined selector , Use to select multiple selector by using ( , ) comma.
